# Changelog

## [0.2.0] - 2026-07-22

Add `Conversations.BulkUploadMemoriesAsync` (POST `/conversations/{id}/memories`)
for bulk uploading a batch of conversation-scoped memories the agent can search
over on demand.

## [0.1.1] - 2026-07-17

Add `Conversations.DeleteAsync` (DELETE `/conversations/{id}`) and
`BackOfficeTasks.DeleteAsync` (DELETE `/back-office-tasks/{id}`) on the Management client.

## [0.1.0] - 2026-06-22

Initial release.
